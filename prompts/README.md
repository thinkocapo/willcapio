## Usage

Each prompt file follows this format:
- **Date**: When the prompt was created
- **Prompt**: The original question or request
- **Content**: The detailed response and any follow-up information

## Adding New Prompts

When adding new prompts:
1. Create a new file named `prompt{N}.md` where N is the next sequential number
2. Follow the established format
3. Update this README with the new prompt information
